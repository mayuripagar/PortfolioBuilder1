# Portfolio Builder (ASP.NET Core MVC)

##  Description
Portfolio Builder is a web-based application developed using ASP.NET Core MVC. 
It allows users to create, manage, and display their professional portfolio in a structured and dynamic way. 
The application follows MVC architecture and demonstrates clean separation of concerns.

##  Features
- User registration and profile management  
- Create and update portfolio details  
- Dynamic data handling using MVC pattern  
- CRUD operations (Create, Read, Update, Delete)  
- Form validation and error handling  

##  Technologies Used
- ASP.NET Core MVC  
- C#  
- MySQL
- Entity Framework Core  
- HTML, CSS, Bootstrap  

## Project Architecture
- Models – Handles data and business logic  
- Views – UI layer for displaying data  
- Controllers – Handles user requests and responses  

##  Key Concepts Used
- MVC Design Pattern  
- Entity Framework Core (ORM)  
- Dependency Injection  
- Razor Views
